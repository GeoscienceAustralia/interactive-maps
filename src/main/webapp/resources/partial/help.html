1.0 What is Interactive Maps?

Interactive Maps is a discovery and exploration view of Geoscience Australia's geospatial services. The following scientific and decision support themes have curated content comprised of maps and functions. Each map has queries and functions with linked access to OGC (Open Geospatial Consortium) web services and metadata. This system replaces MapConnect and AMSIS applications.

1.1 Browser compatibility

The application was tested using the following browser/operating system combinations. Allow cookies and enable JavaScript in the browser to enable full functionality.

  • Firefox version 27 or above
  • Chrome version 32 or above
  • Internet Explorer 9 or above
  • Safari 7 or above

2.0 How to navigate

The application explores cartographic representations of datasets on a map at different scales. The site is layered into themes and maps. Commence by selecting a particular theme, then a map collection.

2.1 Themes and Maps

Geoscience Australia has a selection of themes and maps to explore. Each theme focuses on a particular set of data and has a description of the collection. The maps are curated into sets of complimentary geospatial datasets. To navigate between themes and maps, the top level represents the themes and selecting a theme displays the maps.

When within a map, select the Maps tool on the left side of the map and select though the themes and maps. Alternatively, the Related maps link at the top of the map display maps from the theme. Selecting Home will return the browser to the themes page. Alternatively, the back button on the browser will return to the previous page.

2.2 Pan and Zoom

The Zoom control, right side on the map, allows users to zoom in or out of the map to see greater or less detail. Dragging the slider up or using the top arrow will zoom in on the map. Dragging the slider downwards, or using the bottom arrow, will zoom out. Alternatively, the mouse scroll wheel will zoom the map in or out.

The map can also be panned or moved by clicking on the map, holding down the mouse key and dragging to where you wish to view. Alternatively, the Reference Map (bottom right) can be clicked or dragged to a new location on the map.

2.3 Place name search

Enter the search string into the Place name search box, top right of page, to find place names. Search results will appear down the right side and locations displayed on the map. By selecting a name in the list, the map will display the location.

2.4 Change Basemap

To change the Basemap, select the basemap dropdown box on the map at the top right. If there is no dropdown shown, there are no other Basemaps to select for this map.

2.5 Reset

The Reset feature, activated above the map right side, will reset the map and layers back to the original application defaults. This includes layer visibility, layer transparency, layer order and map extent.

2.6 Keep Extent

Enabling Keep Extent will cause the current view extent to be kept when navigating between different maps.

3.0 How to display map information

Each map application displays a set of cartographic data. Each map contains descriptive, symbolic and metadata information.

3.1 About

The About panel, activated above the map left side, describes the map application currently selected. It contains information about the data in the map and gives context to the collection. The about panel may also contain links to further reading on the map data.

3.2 Legend

The Legend panel, activated above the map left side, gives information on the symbols used on the map. The legend may be a list of links to legend image documents.

3.3 Maps

The Maps panel displays the themes, then maps. Use this to navigate to another theme within the map.

4.0 How to switch map layers

The layers list allows the application to be customised, displaying the layers in different ways.

4.1 Layers control

The Layers panel, activated above the map left side, lists the layers in the map. The layers are initially ordered in a particular way, layered on the map where the top layer is layered at the bottom and the bottom layer is on the top. The layer order can be changed by clicking on a layer, holding down the button down, and then sliding the layer to a new position. The map will change to reflect the new order.

4.2 Metadata

Each layer will have metadata, information on the layer and its history. To access the metadata, click on the Layers control and select the Information icon. The metadata record will appear over the map in a dialog.

5.0 How to use map tools

Some of the maps will have tools that can interact with the map and data. Each map may have a different set of tools available. The tools are listed above the map on the right side next to the Reset button.

5.1 Download dataset

This tool displays links to download datasets that are referenced in, or related to, the map.

5.2 Add interactive map layers

This tool lets users add layers to the current map that are registered with interactive maps. The dialog may have some suggested layers to add, it also has a type box for users to search titles and abstracts.

5.3 Print to PDF

This tool will print the current map to a PDF. The dialog opens with the default title of the map, a selected format (page size) and a checkbox to round the scale. Each map has a scale written, round scale option will adjust the scale of the map to fix a proper map scale.

Click export map button to print the layers into a PDF document. This process may take a few seconds to a minute.

When finished there will be a link to download the PDF.

NOTE: Print to PDF currently does not support drawing graphics.

5.4 Identify features, Legal interests

To Identify features or Interrogate legal interests on the map, click the above the map on the right side. A dialog will open on the right side of the map; click on the map to identify features at the location under the mouse cursor. When complete, the tool will list features, and each of their attributes, at the selected location.

5.5 Measure distance

To Measure distance between two points, select the button above the map on the right side. A panel will open with instructions how to use the tool. To measure, click a point on the map, then another point on the map to which the distance will be measured. By clicking several points on the map, the tool will track each point and each distance and also keep a total distance. To stop the measure tool, double click on the map.

5.6 Distance to boundary

This tool is a custom tool designed to find the shortest perpendicular Distance to a boundary on the map. By clicking the tool above the map on the right side, the panel will open on the right side of the map. Click on the map location to calculate the distance from, in the panel select the boundary type to calculate. Click Search distance to boundary button to run the tool. After a short amount of time, the result will appear in panel and a line to the nearest boundary will be shown on the map.

5.7 Drawing tools

This provides a set of tools to draw graphics on the map. In the dialog, each tool can be used by selecting the tool in the dialog and then interacting with the map. To remove graphics, select the Remove Feature tool and select the graphic.
